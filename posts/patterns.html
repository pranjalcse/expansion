Pattern Printing in Java

Pattern printing in Java is more than just a fascinating coding exercise; it serves as a gateway to enhancing your programming skills and developing a deeper understanding of loops. If you aspire to excel in coding interviews conducted by multinational corporations (MNCs) or simply wish to strengthen your grasp of fundamental concepts, pattern printing is a crucial area to explore. Frequently asked in MNC coding rounds, these questions offer a unique opportunity to showcase your problem-solving abilities while unraveling the inner workings of loops. In this comprehensive blog, we will dive into the captivating world of pattern printing in Java, empowering you to master this skill, unravel the magic of loops, and unleash your coding potential. Get ready to embark on a transformative journey of pattern printing!

To provide structure and clarity, we will divide patterns into two distinct types: 1D patterns and 2D patterns.

1D patterns

To create a 1D pattern in Java, follow these three steps:

  • Determine the pattern and its length.
  • Use a loop to iterate over the desired range.
  • Print each element of the pattern within the loop.

Let's have a look at some examples to help you understand 1D patterns:

Example 1: Printing "*" in a line


public class Example{
    public static void main(String[] args) {
        int length = 5; // Number of asterisks to be printed
        for(int i = 0; i < length; i++) {
            System.out.print("*");
        }
    }
}                            

Output:
*****

In this example, we use a loop to iterate desired number of times and print an asterisk on each iteration.

Example 2: Counting in a line


public class Example{
    public static void main(String[] args) {
        int length = 5; // Number of elements to be printed

        for(int i = 1; i <= length; i++) {
            System.out.print(i);
        }
    }
}                        

Output:
12345

In this example, we use a loop to iterate from 1 to the desired length and print the counting numbers on each iteration.

Example 3: Printing alphabets in a line


public class Example{
    public static void main(String[] args) {
        int length = 5; // Number of alphabets to be printed
        char currentChar = 'a'; // Starting alphabet

        for(int i = 0; i < length; i++) {
            System.out.print(currentChar);
            currentChar++; // Incrementing to the next alphabet
        }
    }
}                      

Output:
abcde

In this example, we use a loop to iterate the desired number of times and print the corresponding alphabets by incrementing the currentChar variable.

2D patterns

To make studying 2d patterns easier, we can further divide it into 3 categories

  • Start Patterns
  • Number Patterns
  • Character Patterns

Star Patterns

1. Square Pattern:The Square Pattern involves printing a square shape composed of asterisks (*) or any other character.


public class Example {
    public static void main(String[] args) {
        int size = 5; // Size of the square pattern

        for(int i = 0; i < size; i++) {
            for(int j = 0; j < size; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}                       

Output:
*****
*****
*****
*****
*****

This code uses nested loops to iterate over the desired size of the square pattern. On each iteration, it prints an asterisk, creating a row of asterisks. The outer loop controls the rows, while the inner loop controls the columns. The result is a square pattern.

2. Hollow Square Pattern:The Hollow Square Pattern is similar to the Square Pattern but with empty spaces in the interior. 


public class HollowSquarePatternExample {
    public static void main(String[] args) {
        int size = 5; // Size of the hollow square pattern

        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (i == 0 || i == size - 1 || j == 0 || j == size - 1) {
                    System.out.print("*");
                } else {
                    System.out.print(" ");
                }
            }
            System.out.println();
        }
    }
}                   

Output:
*****
*   *
*   *
*   *
*****

In this example, we add a conditional statement within the inner loop to check if the current position is on the border of the square. If it is, we print an asterisk; otherwise, we print a space to create the hollow effect. This condition allows us to selectively print asterisks only at the border positions, leaving the interior empty.

3. Left-Facing Right-Angled Triangle Pattern:The Left-Facing Right-Angled Triangle Pattern involves printing a triangle shape with the longest side on the left side.


public class LeftTrianglePatternExample {
    public static void main(String[] args) {
        int size = 5; // Size of the left-facing right-angled triangle pattern

        for (int i = 0; i < size; i++) {
            for (int j = 0; j <= i; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}                

Output:
*
**
***
****
*****

Using nested loops, we control the number of asterisks printed in each row. The outer loop determines the number of rows, and the inner loop determines the number of asterisks in each row. As the outer loop variable (i) increases, the number of asterisks printed in each row also increases, creating a left-facing right-angled triangle shape.

Right-Facing Right-Angled Triangle Pattern:The Right-Facing Right-Angled Triangle Pattern is similar to the left-facing triangle but with the longest side on the right side.


public class RightTrianglePatternExample {
    public static void main(String[] args) {
        int size = 5; // Size of the right-facing right-angled triangle pattern

        for (int i = 0; i < size; i++) {
            for (int j = size - i; j > 1; j--) {
                System.out.print(" ");
            }
            for (int k = 0; k <= i; k++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}           

Output:
    *
   **
  ***
 ****
*****

In this code, we use two nested loops. The outer loop controls the rows of the triangle, while the inner loops control the spaces and asterisks in each row.

The first inner loop (for (int j = size - i; j > 1; j--)) is responsible for printing the spaces before the asterisks. It starts with size - i, which determines the number of spaces to be printed. As i increases (outer loop variable), the number of spaces decreases.

The second inner loop (for (int k = 0; k <= i; k++)) prints the asterisks. It iterates from 0 to i (inclusive) to determine the number of asterisks in each row. As i increases, the number of asterisks printed in each row also increases.

By combining these loops and print statements, we achieve the right-facing right-angled triangle pattern, where the longest side is on the right side.

Character patterns

1. Square Pattern:


public class CharacterSquarePatternExample {
    public static void main(String[] args) {
        int size = 3; // Size of the square pattern
        char currentChar = 'A'; // Starting character

        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                System.out.print(currentChar);
                currentChar++; // Incrementing to the next character
            }
            System.out.println();
        }
    }
}

Output:
ABC
DEF
GHI

The square character pattern creates a square shape using characters. Each row or column represents a sequence of characters. In this example, we use nested loops to print a square pattern. The outer loop controls the number of rows, and the inner loop controls the number of characters in each row. By using the currentChar variable and incrementing it in each iteration, we print consecutive characters in each position.

Complex patterns

In the case of complex patterns, it can be beneficial to break them down into several simpler smaller patterns. By understanding and recreating these smaller patterns, we can gradually build towards the complex pattern we desire. Breaking down the complex pattern into simpler components allows us to tackle each component individually, making it easier to comprehend and implement. Once we have created and understood the smaller patterns, we can then combine them strategically to reconstruct the complex pattern. This approach not only simplifies the overall problem but also helps in better organizing our code and improving code readability.

  • Understand the pattern: Analyze the arrangement and rules.
  • Plan your approach: Decide on the looping structure and logic.
  • Break it down: Simplify complex patterns into smaller components.
  • Use nested loops: Control rows and columns or elements.
  • Manage spacing: Add gaps or alignment where necessary.
  • Track element values: Increment or modify values accurately.
  • Test and iterate: Check output, make adjustments if needed.
  • Maintain code readability: Use clear variable names and comments.

Following these guidelines ensures successful pattern printing. Practice and experimentation are key to improving your skills.
