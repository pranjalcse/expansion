Unary Operators

Unary operators are operators that work on a single operand, which can be a variable, constant, or expression. Unary operators can be used to perform various operations such as incrementing or decrementing a value, changing the sign of a value, or performing logical negation.

Types of Unary Operators:

There are six types of unary operators in Java, namely:

  1. Unary Plus Operator(+)
  2. Unary Minus Operator(-)
  3. Increment Operator(++)
  4. Decrement Operator(--)
  5. Logical Compliment Operator(!)
  6. Bitwise Compliment Operator(~)

Unary plus operator (+)

The unary plus operator is represented by a plus symbol (+) and it simply returns the value of the operand. It is used to explicitly show the positive value of a variable or expression, although this is often redundant since positive values are already assumed in Java.


public class Example {

    public static void main(String[] args)
    {
        int a = 5;
        int b = +a; //Unary plus operator
        System.out.println("Value of b is " + b);
    }
}

Output:Value of b is 5

Unary minus operator (-)

The unary minus operator is represented by a minus symbol (-) and it returns the negative value of the operand. It is used to negate a number or expression.


public class Example {

    public static void main(String[] args)
    {
        int a = 5;
        int b = -a;
        System.out.println("Value of b is " + b);
    }
}

Output:Value of b is -5

Increment operator (++)

The increment operator is represented by two plus symbols (++) and it increases the value of the operand by 1. It is used to update a variable with a new value.


public class Example {

    public static void main(String[] args)
    {
        int a = 5;
        a++;
        System.out.println("Value of a is " + a);
    }
}

Output:Value of a is 6

Decrement operator (--)

The decrement operator is represented by two minus symbols (--) and it decreases the value of the operand by 1. It is used to update a variable with a new value.


public class Example {

    public static void main(String[] args)
    {
        int a = 5;
        a--;
        System.out.println("Value of a is " + a);
    }
}

Output:Value of a is 4

Logical complement operator (!)

The logical complement operator is represented by an exclamation mark (!) and it inverts the boolean value of the operand. It is used to check if a boolean value is false.


public class Example {

    public static void main(String[] args)
    {
        boolean a = true;
        boolean b = !a;
        System.out.println("Value of b is: " + b);
    }
}

Output:Value of b is false

Bitwise complement operator (~)

The bitwise complement operator is represented by a tilde symbol (~) and it flips the bits of the operand. It is used to perform bitwise operations.


public class Example {

    public static void main(String[] args)
    {
        int a = 5;
        int b = ~a;
        System.out.println("Value of b is: " + b);
    }
}

Output: Value of b is: -6

Note that the output of the bitwise complement operator in the above example is -6 because the bitwise representation of 5 is 00000101 in binary and the bitwise complement of that is 11111010, which is the two's complement representation of -6.

Understanding Increment and Decrement Operators in Java: Prefix vs Postfix

Increment and decrement operators (++ and --) can be used in two ways:

  1. Pre-increment/decrement
  2. Post-increment/decrement

Pre-increment/decrement

The value of the variable is first incremented or decremented, and then the new value is used in the expression.


public class Example {

    public static void main(String[] args)
    {
        int x = 5;
        int y = ++x; // x will be 6, y will also be 6
        System.out.println(x+" "+y);

        int c = 5;
        int d = --d; // d will be 4, c will also be 4
        System.out.println(x+" "+y);

        int a = 5;
        System.out.print(++a + " ");  // 6 will be printed
        System.out.print(a); // 6 will be printed
    }
}

Output: 
6 6
4 4
6 6

Post-increment/decrement

The value of the variable is first used in the expression, and then it is incremented or decremented. 


public class Example {

    public static void main(String[] args)
    {
        int x = 5;
        int y = x++; // x will be 6, y will be 5
        System.out.println(x+" "+y);
        
        int c = 5;
        int d = c--; // c will be 4, d will be 5
        System.out.println(c+" "+d);

        int a = 5;
        System.out.print(a++ +" ");  // 5 will be printed
        System.out.print(a);    // 6 will be printed
    }
}

Output: 
6 5
4 5
5 6
